**项目目标**
------------

*   使用 LLM OpenAI API 统一模型，集成 QWEN 和 OpenAI-4o-Mini 。
*   实现流程更清晰的个人传记编写系统，分为两大部分：(解除耦合，先保存采集数据，再进行编写)
    *   **收集资料**：采集音频与文字稿，整理并数字化保存。
    *   **传记编写内容**：完全体，包括 Agent Writer、Critic、Scorer 等等。
